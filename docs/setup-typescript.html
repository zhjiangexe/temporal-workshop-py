Setup — Node + TypeScript

Workshop SetupNode.jsTypeScript

這份 workshop 需要什麼？

工具本 workshop 用途確認指令
git下載這份 repository。git --version
node / npm安裝與執行 typescript/ 版本。node --version
npm --version
temporal啟動本機 Temporal Server 與 Web UI。temporal --version

建議安裝方式

macOS / Homebrew：

brew install node temporal

安裝完成後確認：

node --version
npm --version
temporal --version

下載並初始化 TypeScript 版

git clone git@github.com:zhjiangexe/temporal-workshop-py.git
cd temporal-workshop-py/typescript
npm install
注意
TypeScript 版本獨立放在 typescript/。所有 npm 指令都在這個資料夾執行。

確認 TypeScript 版可以跑

npm run typecheck
npm test

啟動 Temporal Server

開一個 terminal，保持它執行：

temporal server start-dev
項目位置
Worker / Starter 連線位置localhost:7233
Web UIhttp://localhost:8233

跑第一個流程

在 typescript/ 目錄，開第二個 terminal 啟動 Worker：

npm run process1:worker

開第三個 terminal 啟動 Workflow：

npm run process1:starter

本 workshop 的 TypeScript 指令

流程WorkerStarter / Update
Process 1npm run process1:workernpm run process1:starter
Process 2npm run process2:workernpm run process2:starter
Process 2 失敗示範FAIL_PUBLISH=1 npm run process2:workernpm run process2:starter
Process 3npm run process3:workernpm run process3:starter
npm run process3:update -- <WorkflowId>
Workshop 原則
TypeScript 版只涵蓋 Process 1～3，且放在 typescript/，不要和 Python 的 python/src/temporal_learn/ 混在一起執行。
